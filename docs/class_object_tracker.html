bool ObjectTracker::confirmTracker ( )

Method used to confirm that the tracker is still accurately tracking a face. Checks the last 100 stored tracking coordinates to make sure that the tracker is not just sitting still. If an object is still being accurately tracked, there will be enough variation in the bounding box for this method to return true.

Returns
bool indicating whether or not the tracker is still accurate. False indicates that the tracker should be destroyed and reinitialized
